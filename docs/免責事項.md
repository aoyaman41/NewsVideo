# 免責事項

このドキュメントは、NewsVideo の `OSS Public Beta` を利用する前に共有しておきたい前提を、ユーザー向けに短くまとめたものです。

## ソフトウェアの提供形態

- NewsVideo は個人開発の `OSS Public Beta` です
- ソフトウェアは `AS IS` で提供します
- 動作継続性、特定用途への適合性、生成結果の品質、将来の互換性は保証しません
- 外部 API の仕様変更、障害、レート制限、料金改定の影響を受けます

法的な無保証と責任制限の本文は [LICENSE](../LICENSE) を確認してください。

## サポート範囲

- サポートは `best effort` です
- 個別環境ごとの無償セットアップ支援、運用代行、商用 SaaS 相当の継続サポートは行いません
- 署名済み配布や notarization が必須の環境では、現時点の Public Beta は適していません

## 費用と外部サービス

- API 利用料は NewsVideo ではなく、ユーザー自身の OpenAI / Google AI 契約に対して直接発生します
- 外部 API やモデルの利用条件は、それぞれの提供元の規約に従ってください

## 公開デモ素材について

- README、GitHub Releases、docs に載せている公開用デモ素材は、このリポジトリ用に作成した架空記事とサンプルデータを使っています
- `examples/public-beta-sample.newsproj` の記事入力は `架空ニュース通信` 名義の架空記事で、第三者記事の転載を意図していません
- `.github/assets/public-beta/sample-output/` の見本画像は、外部画像を取り込まず、リポジトリ内スクリプトで描画した図版です
- 確認した範囲では、公開デモ素材に第三者ロゴ、写真、透かし、引用記事本文は含まれていません

## 判断に迷う場合

- 管理対象端末での実行、社内規程との整合、知財・法務判断が必要な用途では、利用者側で確認してください
- このドキュメントは公開前提の整理であり、法的助言ではありません
