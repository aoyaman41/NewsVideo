# サンプルプロジェクト

このドキュメントは、記事入力済みの公開用サンプル `.newsproj` を使って、NewsVideo の最小フローを試すための手順です。

- このサンプル `.newsproj` に含まれる記事入力は公開用の架空データです
- 画像、音声、出力ファイルは同梱しておらず、第三者メディア素材の配布を意図していません

## どこにあるか

- サンプル本体: `examples/public-beta-sample.newsproj`

このサンプルには、[サンプル記事](./サンプル記事.md) と同じ記事入力が最初から入っています。`parts`、`prompts`、`images`、`audio` は空なので、スクリプト生成から先をそのまま試せます。

## 使い方

1. NewsVideo を一度起動して終了します
2. `examples/public-beta-sample.newsproj` を、アプリの `projects` ディレクトリにコピーします
3. NewsVideo を再起動します
4. プロジェクト一覧に `Public Beta Sample` が出てきたら開きます

## コピー先の目安

### 配布版を使っている場合

- 目安: `~/Library/Application Support/NewsVideo/projects/`

### 開発版を使っている場合

- `NEWSVIDEO_USER_DATA_DIR` を使っているなら、その配下の `projects/`
- 使っていない場合は、Electron の `userData/projects/` 配下

補足:

- コピー先が分からない場合は、まず新規プロジェクトを 1 つ作成し、その後に作られた `*.newsproj` と同じ階層に置いてください
- フォルダ名の `.newsproj` 拡張子は消さないでください

## このサンプルで試せること

- プロジェクト一覧に表示されること
- 記事画面でタイトル、出典、本文が保存済みであること
- スクリプト生成から画像、音声、動画へ進めること

## 収録内容

- `project.json`
- `article.json`
- `parts.json`
- `images.json`
- `prompts.json`
- `audio.json`
- `usage.json`

画像、音声、動画の実ファイルは同梱していません。
