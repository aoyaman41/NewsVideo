# 6. 画面設計書

## 6.1 画面一覧

| 画面ID | 画面名 | パス | 説明 |
|--------|--------|------|------|
| SCR-001 | プロジェクト一覧 | `/` | プロジェクトの一覧表示・作成 |
| SCR-002 | 記事入力 | `/project/:id` | 記事テキスト・画像の入力 |
| SCR-003 | スクリプト編集 | `/project/:id/script` | パート・スクリプトの編集 |
| SCR-004 | 画像管理 | `/project/:id/images` | 画像生成・割り当て管理 |
| SCR-005 | 動画プレビュー | `/project/:id/preview` | プレビュー・動画出力 |
| SCR-006 | 設定 | `/settings` | API設定・デフォルト設定 |

### 6.7.2 機能要素

| 要素 | 説明 |
|------|------|
| 動画プレビューエリア | 動画をプレビュー再生 |

| タイムライン | 全体構成を視覚的に表示、クリックで該当位置へジャンプ |
| 画像切替ポイント（▼マーク） | タイムライン上に画像切替タイミングを表示、クリックで該当画像へジャンプ |

| 音声未生成警告バッジ（⚠） | 音声が未生成のパートに警告を表示 |
| 出力設定 | 解像度、FPS、ビットレートを設定 |
| 出力先選択 | 出力ファイルパスを指定 |
| 書き出しボタン | 最終動画をレンダリング |
