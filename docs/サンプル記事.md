## フィールド別（コピペ用）

### タイトル

```text
再エネ拡大と蓄電池導入、来夏の電力需給を左右　系統混雑も課題
```

### 出典

```text
架空ニュース通信（2025-12-10）
```

### 本文

```text
経済産業省は10日、来夏（2026年）の電力需給見通しを公表した。猛暑を想定したピーク時の需要は全国で前年比2%増を見込み、供給余力は地域差が大きいという。太陽光・風力など再生可能エネルギーの導入拡大が進む一方、出力変動に対応するための蓄電池や需給調整力の確保が重要になる。

見通しでは、昼間の太陽光発電が増えることで一部地域では余剰が発生しやすい反面、夕方以降は需要が急増し、火力の立ち上げや蓄電池の放電が不足すると価格が急騰する可能性がある。昨夏のスポット市場では夕方帯に価格が一時的に高騰した日が複数あった。

地域別では、九州や北海道などで系統混雑が起きやすく、発電しても送電容量の制約で出力抑制が増える見込みだ。広域連系線の増強や、需要地近くでの蓄電池設置、企業によるデマンドレスポンス（節電要請への協力）を組み合わせ、無駄なく電力を使う必要がある。

国の中期目標では、2030年度の電源構成に占める再エネ比率を36〜38%程度とする方針だ。関係者は「再エネの比率が高まるほど、調整力（蓄電池・揚水・火力の柔軟運転・需要側）の価値が上がる」と指摘する。

今後の焦点は①蓄電池の導入スピード、②系統増強の着工と完成時期、③需要家の参加（ピークカット・時間帯別料金の活用）だ。政府は設備投資の支援策や市場設計の見直しを進め、需給の安定化と電気料金の抑制につなげたい考えだ。
```
